Introduction


What is C++?

C++ is a general-purpose, high-performance programming language that is widely used for system and application software, game development, real-time simulations,
and performance-critical applications. It was created by Bjarne Stroustrup in 1979 as an extension of the C programming language. C++ introduces object-oriented
programming (OOP) concepts like classes, inheritance, and polymorphism, which helps in organizing and managing large and complex software projects.


Some key features of C++ include:

1. Object-Oriented Programming (OOP): It supports classes, objects, inheritance, polymorphism, and encapsulation, making it easier to model complex systems.

2. Memory Management:C++ gives developers control over memory allocation and deallocation through pointers, which allows for more efficient use of memory.

3. Low-Level Manipulation:C++ allows direct manipulation of hardware and memory, which is useful for system-level programming.

4. Performance:It is known for its speed and efficiency, making it suitable for performance-critical applications like game engines and high-frequency trading platforms.

5. Standard Template Library (STL):C++ provides a collection of template classes and functions (like vectors, maps, and algorithms) that simplify common tasks.

Lesson-3: conditional statement

if-esle statement

The if-else statement in C++ is used for decision-making.If the condition evaluates to true, the code inside the if block is executed; otherwise, the code inside the else block runs.

Basic Syntax:

                if (condition) {
                    // Code to execute if condition is true
                } else {
                    // Code to execute if condition is false
                }

Example code:

                #include 
                    using namespace std;
                    
                    int main() {
                        int number;
                        cout << "Enter a number: ";
                        cin >> number;
                    
                        if (number > 0) {
                            cout << "The number is positive." << endl;
                        } else {
                            cout << "The number is negative or zero." << endl;
                        }
                    
                        return 0;
                    }

Explanation:

1. Condition: The expression number > 0 is evaluated.

2. True Case: If the condition is true (the number is greater than zero), it prints "The number is positive."

3. False Case: If the condition is false (the number is zero or negative), it prints "The number is negative or zero."

Extended If-Else (Using else if):

You can also use multiple conditions with else if to check for more than two possibilities.

Example code:

                #include 
                    using namespace std;
                    
                    int main() {
                        int number;
                        cout << "Enter a number: ";
                        cin >> number;
                    
                        if (number > 0) {
                            cout << "The number is positive." << endl;
                        } else if (number < 0) {
                            cout << "The number is negative." << endl;
                        } else {
                            cout << "The number is zero." << endl;
                        }
                    
                        return 0;
                    }

Explanation:

1. If number > 0, the program prints that the number is positive.

2. If number < 0, it prints that the number is negative.

3. If the number is neither (i.e., zero), it prints "The number is zero."

Nested If-Else Statement

A nested if-else statement in C++ is when one if-else statement is placed inside another if-else statement. This allows you to check multiple conditions and
perform more complex decision-making.

Example code:

                    #include 
                        using namespace std;
                        
                        int main() {
                            int number;
                            cout << "Enter a number: ";
                            cin >> number;
                        
                            if (number > 0) { // First condition: check if number is positive
                                cout << "The number is positive." << endl;
                        
                                if (number % 2 == 0) { // Nested condition: check if number is even
                                    cout << "The number is even." << endl;
                                } else { // If number is odd
                                    cout << "The number is odd." << endl;
                                }
                        
                            } else if (number < 0) { // Second condition: check if number is negative
                                cout << "The number is negative." << endl;
                            } else { // Third condition: number is zero
                                cout << "The number is zero." << endl;
                            }
                        
                            return 0;
                        }

Explanation:

1. First, the program checks if the number is positive using the outer if (number > 0).

2. If the number is positive, a nested if checks if the number is even or odd using number % 2 == 0.

3. If the number is not positive, it checks if it's negative using else if (number < 0).

4. If neither condition is true, it means the number is zero, and the final else block is executed.
